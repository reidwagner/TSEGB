# TSE Z80 Emulator

Implemented instructions, with r being any 8-bit register:
- NOP
- ADD A, r
- ADC A, r
- AND A, r
- XOR A, r
- OR A, r
- LD r, n
- HALT

The [zasm](https://k1.spdns.de/Develop/Projects/zasm/Distributions/) assembler can be used to assemble the programs in assembly/src/.
